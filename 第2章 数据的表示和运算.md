* [Y]补 = [-Y]补 整体取反+1（连同符号位）
  ```mermaid
  erDiagram
      %% 实体定义（格式：ENTITY { type fieldName PK/FK }）
      CUSTOMER {
          int customer_id PK
          string name
          string email
          string phone
      }

      ORDER {
          int order_id PK
          date order_date
          float total_amount
          int customer_id FK
      }

      PRODUCT {
          int product_id PK
          string name
          float price
          string category
      }

      ORDER_ITEM {
          int order_item_id PK
          int order_id FK
          int product_id FK
          int quantity
          float item_price
      }

      SUPPLIER {
          int supplier_id PK
          string name
          string contact
      }

      PRODUCT_SUPPLIER {
          int product_id FK
          int supplier_id FK
      }

      %% 关系（cardinality 用 Mermaid 的语法表示）
      CUSTOMER ||--o{ ORDER : "places"
      ORDER ||--|{ ORDER_ITEM : "contains"
      PRODUCT ||--o{ ORDER_ITEM : "included in"

      %% 多对多通过关联表 PRODUCT_SUPPLIER
      PRODUCT ||--o{ PRODUCT_SUPPLIER : "supplied by"
      SUPPLIER ||--o{ PRODUCT_SUPPLIER : "supplies"

      %% 可选：如果想显示外键指向，下面这行可强化语义（Mermaid 还是以关系线为主）
      ORDER }o--|| CUSTOMER : "customer_id -> customer_id"

  ```
