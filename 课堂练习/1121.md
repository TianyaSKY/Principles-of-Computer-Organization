# 知识点

TPI 是 **Tracks Per Inch** 的缩写，中文意思是“每英寸磁道数”，它是硬盘（尤其是传统机械硬盘，HDD）中的一个技术参数，用来描述磁盘表面磁道密度的物理指标。

| 缩写          | 名称            | 含义                                                                     |
| ------------- | --------------- | ------------------------------------------------------------------------ |
| TPI           | Tracks Per Inch | 每英寸磁道数，径向密度                                                   |
| BPI           | Bits Per Inch   | 每英寸位数，沿磁道方向的数据密度                                         |
| DPI           | Dots Per Inch   | 更通用的密度单位，有时也用于描述存储密度                                 |
| Areal Density | 面密度          | = TPI × BPI，单位通常是 Gb/in²（每平方英寸的比特数），反映整体存储密度 |

**传输速率 = 每磁道容量 × 转速 (每秒转数)**

磁盘总容量的计算公式为： 总容量 = 有效记录面数 × 每面磁道数 × 每磁道容量

# 题目

1设磁盘组有17个盘片，每片有两个记录面；存储区域内直径2.36英寸，外直径5.00英寸；道密度为1 250TPI，内层位密度52 400bpi，转速为2 400rpm。(TPI表示每英寸磁道数，bpi表示每英寸位数)，问盘组总存储容量。

```markdown
总容量的计算公式为： 总容量 = 有效记录面数 × 每面磁道数 × 每磁道容量
有效记录面数 = 17 * 2 - 2 = 32
每面磁道数 =  1250 * (5.00-2.36)/2 = 1650
每磁道容量 = 最内圈周长 * 内层位密度 = (pi * 2.36) * 52400 = 388,503 bits
388503*1650*32/8/1024/1024/1024=2.39G
```

2CRT分辨率为1024*1024像素，像素的颜色数为256，则刷新存储器容量为

```markdown
256 颜色 -> 2^{8} = 1b
2^{20}b = 1M
```

3设磁盘组有11个盘片，每片有两个记录面；存储区域内直径2.36英寸，外直径5.00英寸；道密度为1 250TPI，内层位密度52 400bpi，转速为2 400rpm。(TPI表示每英寸磁道数，bpi表示每英寸位数)，问盘组总存储容量和数据传输率分别是多少?

```markdown
面数=11*2-2=20
道数=1250*(5.00-2.36)/2=1650
每磁道容量=52400*(2.36*pi)=388503
ans=1.49GB
2400rpm=40rps
传输速率=40*388503/8/1024/1024=1.85
```

磁盘组有6片磁盘，最外两侧盘片**可以记录**，存储区域内径22cm，外径33cm，道密度为40道/cm，内层密度为400位/cm，转速为3600转/分。盘组总存储容量是多少？

```markdown
面数=6*2=12
每面道数=(33-22)*40/2=220道
每道容量=220*400*pi=27640.1535

12 * 220 * 27640.1535 => 8.70MB
```

某磁盘存储器转速为3000 rpm，共有4个记录盘面，道密度为5 tpm，每道记录信息12288 B,最小磁道直径为230 mm，共有275道，求：

最高位密度和最低位密度

```markdown
12288B -> 98304 bit
最高位密度出现在最小直径的磁道上。
最高位密度98304/(230pi)=136bit/mm

width = 275/5 = 55mm
最大直径=230+55*2=340mm
98304 / 340pi = 92bit/mm
```

显示器的屏幕有24行，每行80个汉字，假设汉字字模采用16×16点阵，请问保存一帧信息所需要的存储空间。

```markdown
24×80×2^{8}/8=60KB
```
